# GoResumeReview 开发规划

## 项目概述

GoResumeReview 是一款面向 HR 群体的智能简历审核工具，支持 Windows 桌面客户端，采用 Go + Vue3 技术栈开发，实现高并发简历分析和可视化预览功能。

## 开发目标

### 核心目标
1. **易用性**：零学习成本，HR 无需技术背景
2. **高效性**：批量处理，高并发分析
3. **准确性**：AI 驱动，精准评估
4. **可扩展**：插件化架构，易于扩展

### 非目标
- 不追求功能复杂
- 不支持移动端
- 不做在线版本（纯本地应用）

## 技术决策

### 已决定

| 决策点 | 选择 | 理由 |
|--------|------|------|
| GUI 框架 | Wails v2 | Web 技术栈，现代 UI |
| 前端框架 | Vue3 + TS | 轻量、响应式 |
| UI 库 | Element Plus | 成熟、HR 熟悉 |
| PDF 解析 | pdfcpu | 开源、活跃维护 |
| Word 解析 | docxgo | MIT 许可、纯 Go |
| OCR | gosseract | Tesseract 封装 |
| 数据库 | SQLite | 轻量、无需服务 |

### 待决定

| 决策点 | 选项 | 评估标准 |
|--------|------|----------|
| OCR 引擎 | Tesseract / PaddleOCR | 准确率对比 |
| 安装包 | NSIS / Inno Setup | 兼容性、易用性 |
| CI/CD | GitHub Actions | 免费、集成度 |

## 风险识别

| 风险 | 可能性 | 影响 | 应对措施 |
|------|--------|------|----------|
| OCR 准确率不足 | 中 | 高 | 支持多引擎切换 |
| AI API 限流 | 中 | 中 | 实现请求队列和重试 |
| 大文件解析慢 | 低 | 中 | 异步处理、进度反馈 |

## 质量标准

### 代码质量

- Go 代码：golint 通过率 100%
- Vue 代码：ESLint 通过率 100%
- 测试覆盖率：核心模块 > 80%

### 性能标准

- 简历解析：单份 < 3 秒
- AI 分析：单份 < 10 秒
- 批量处理：10 份简历 < 60 秒
- 内存占用：< 200MB

### 用户体验

- 首屏加载：< 3 秒
- 操作步骤：最少 3 步完成分析
- 错误提示：清晰、友好
